### run mybugfinder on Joda
1. Put joda project in the same directory as mybugfinder.py and analysis.py
2. Install javalang library for python in your local machine.
3. Run "python mybugfinder.py" and you will see all the file being processed and errors found below the cooresponding files.

### test
1. Put the 6 testpattern*-*.java in the same directory as other dependencies.
2. Run "python testpatterns.py"
